梅露可圖鑑

梅露可圖鑑是一款用於查看梅露可物語，同伴、魔寵立繪與數據的App。

能夠對傭兵、魔寵列表進行排序、篩選；立繪支持雙指放大，立繪界面右劃可以看到詳細信息！
